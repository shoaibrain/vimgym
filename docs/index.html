What it does

Every Claude Code session is captured automatically the moment it lands on disk, indexed with full-text search, and made searchable from a local web UI or your terminal. No accounts, no telemetry, no upload. The vault never leaves your machine.

⊕ Search

Full-text search across every prompt, response, tool use, and file edit. Find the session where you decided how the auth flow worked.

⊞ Three-pane UI

Inbox of sessions, sidebar of projects and branches, full conversation rendering with syntax highlighting and copy-to-clipboard.

● Live capture

A filesystem watcher catches every new session within seconds. Restart your laptop, lose nothing. Sessions persist forever.

Install

Three ways. Pick the one that matches your setup.

homebrew

$ brew install vimgym
$ vg start

curl

$ curl -fsSL \
  https://vimgym.xyz/install | sh

pipx

$ pipx install vimgym
$ vg init && vg start

Requires Python 3.11+. macOS and Linux supported. Windows support in v2.

Sources

vimgym auto-detects every AI coding tool installed on your machine. Only Claude Code has a parser in v1; other tools are detected and listed but disabled until their parser ships.

Privacy

vimgym binds to 127.0.0.1 only. The vault is a SQLite file at ~/.vimgym/vault.db with 0600 permissions. A built-in redaction engine scrubs API keys, AWS credentials, kubeconfig certs, JWT tokens, database URLs, PEM blocks, and npm tokens before anything is written. No telemetry. No cloud. No outbound network calls except for the daemon's own filesystem watcher.
