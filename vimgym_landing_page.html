how it works
zero config. just runs.
01 — WATCH
you code, we watch
vimgym watches ~/.claude/projects/ for new session files. No hooks. No config. No interruptions.
→
02 — CAPTURE
session goes quiet
5-second debounce. Stability check. Parse the JSONL. Strip secrets with 18 redaction patterns. Never reads raw credentials.
→
03 — INDEX
straight to sqlite
FTS5 full-text index. BM25 ranking. Your messages, the AI's responses, every tool call — all searchable in milliseconds.
→
04 — SEARCH
find anything
vg search "auth flow" or press ⌘K in the browser UI. Your entire AI history, searchable instantly.
features
built by developers, for developers
vg

fully local, zero cloud

Your sessions never leave your machine. SQLite database at ~/.vimgym/vault.db. Works offline. Always.

18

redaction by default

18 patterns strip API keys, AWS credentials, JWT tokens, kubeconfig certs, SSH keys before anything hits disk. Your secrets stay secret.

⌘K

neon void UI

Three-pane browser UI. Command palette. Session detail with syntax highlighting. Vim Airline status bar. JetBrains Mono everywhere.

↓

export to resume

Export any session as Markdown and paste it back into Claude Code. Never lose your context again — even if the session crashes.

ws

live updates

WebSocket pushes new sessions to the UI in real time. Session card appears and flashes green within 5 seconds of Claude Code going quiet.

5x

multi-source ready

Detects Claude Code, Cursor, Copilot, Gemini, Antigravity on install. Claude Code in v1. Others in v2. Architecture already supports all.
